# 💵 Conversor de Real para Dólar

## 📖 Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de converter um valor em reais (R$) para dólares (US$), utilizando a cotação informada pelo usuário.

O sistema solicita a cotação atual do dólar e o valor em reais que deseja ser convertido, realizando o cálculo automaticamente e exibindo o resultado na tela.

---

## 🚀 Funcionalidades

* Informar a cotação do dólar;
* Informar o valor em reais;
* Realizar a conversão para dólar;
* Exibir o valor convertido.


---

## 📋 Fórmula Utilizada

A conversão é realizada utilizando a seguinte fórmula:

Valor em Dólar = Valor em Reais ÷ Cotação do Dólar

Exemplo:

```text
5 ÷ 10 = 0.5
```
